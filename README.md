# Анализ конференций TED: Дашборд для запуска франшизы (Yandex DataLens)

![TED Dashboard](assets/image_72e785.png)

## Бизнес-контекст задачи
Заказчик приобрел лицензию на проведение классических конференций TED в России. Для организации первого мероприятия потребовался аналитический инструмент, опирающийся на исторический опыт TED. 
Дашборд помогает определить оптимальный хронометраж, наиболее востребованные у аудитории темы и профили спикеров, вызывающих максимальный отклик.

## Описание данных
Источником выступает реляционная база открытых данных TED, состоящая из 3 связанных таблиц:
* `events` - справочник проведенных конференций (география, названия).
* `speakers` - профили выступающих (род деятельности, описание).
* `talks` - логи выступлений (просмотры, хронометраж, теги, реакции аудитории: смех и аплодисменты).

## Функционал и метрики дашборда
* **Верхнеуровневые KPI:** Мониторинг общего объема базы для оценки релевантности выборки (количество конференций, выступлений, уникальных спикеров и тем).
* **Анализ контента:** Выявление топ-20 популярных тегов и рейтинга самых смешных выступлений по уровню реакции зала.
* **Портрет спикера:** Оценка вовлеченности аудитории (аплодисменты) в разрезе профессий выступающих.
* **Организационные метрики:** Сводная детализация по странам и конференциям для определения стандарта проведения (средняя длительность выступлений, максимумы просмотров).
* **Интерактивность:** Реализована гибкая система фильтрации (селекторы по стране, названию конференции, тегу и дате). Добавлена сквозная таблица детализации для просмотра каждого отдельного выступления.
